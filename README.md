# LLM4XR-Perf
 A Benchmark and Empirical Evaluation of Line-level Localization for VR/AR Performance Defects

## Set up
`pip install -r requirements.txt`

Set the BASE_URL. API_KEY, GITHUB_TOKEN in the `inference.py` and `evaluation.py`. Set the LLM at the same time.

## Quick Start
Download the dataset from [Zenodo](https://zenodo.org/records/17898806) and put it in the `/data` path.

### Inference
`python inference.py`

### Evaluation
`python evaluation.py`
